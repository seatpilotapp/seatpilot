## 11. Guías operativas de apoyo

### 11.1 Opciones de mesas (referencia rápida)

| Tipo de mesa | Dimensiones | Capacidad sugerida |
| --- | --- | --- |
| Cocktail (pedestal) | 24″ / 30″ / 36″ Ø | 2–6 personas de pie |
| Banquet estándar | 6′ × 30″ | 6–8 personas |
| Banquet extendida | 8′ × 30″ | 8–10 personas |
| Redonda pequeña | 36″ Ø | 4–6 personas |
| Redonda mediana | 48″ Ø | 6–8 personas |
| Redonda grande | 60″ Ø | 8–10 personas |
| Redonda XL | 72″ Ø | 10–12 personas |
| Mesa infantil | 24″ × 48″ | 6–8 niños |

### 11.2 Fórmulas de pasillos y densidad

- **Banquet style:** 24″ para servicio + 18″ profundidad silla.  
- **Round tables:** dejar 54″ entre mesas para flujo y servicio. Ocupación rápida ≈ área (ft²) ÷ 10; agrega 2 ft² por persona para mayor comodidad.  
- **Oblong seating:** 60″ entre mesas back-to-back. Área ÷ 8 → máximo teórico, suma 2 ft²/persona para comodidad.  
- **Classroom:** 36″ entre mesas (una cara). Incrementa para doble cara o servicio.  
- **Theater style:** 36″ entre filas de sillas; área ÷ 8 ≈ capacidad.  
- **Dance floor:** calcula 2–4 ft² por persona según nivel de baile esperado.

### 11.3 Guía de mantelería

| Mesa | Half Drop | 3/4 Drop | Full Drop |
| --- | --- | --- | --- |
| Banquet 6′ | 60″ × 108″ | — | 90″ × 120″ |
| Banquet 8′ | 60″ × 126″ | — | 90″ × 156″ |
| Cocktail 36″ Ø | 90″ Ø | 108″ Ø | 120″ Ø |
| Redonda 36″ Ø | 90″ Ø | — | — (full drop con 90″) |
| Redonda 48″ Ø | 90″ Ø | 108″ Ø | — |
| Redonda 60″ Ø | 90″ Ø | 108″ Ø | 120″ Ø |
| Redonda 72″ Ø | 108″ Ø | 120″ Ø | 132″ Ø |
| Servilletas | — | — | 20″ × 20″ |

> Definiciones: **Half drop** = llega a media altura, **¾ drop** = tres cuartos, **Full drop** = toca el piso. Ajusta según protocolo del evento y exposición al viento.

### 11.4 Operación segura de chafing dishes (canned heat)

1. Usa únicamente combustible enlatado aprobado para las charolas.  
2. Llena la bandeja de agua antes de encender el quemador.  
3. Abre el envase de combustible, colócalo en el soporte y enciéndelo con la tapa medio abierta.  
4. Mantén la llama baja para conservar temperatura estable; evita exponer la lata a fuego directo.  
5. Verifica periódicamente el nivel de agua para evitar que se evapore por completo.  
6. Protege manteles y superficies con charolas bajo los quemadores.  
7. No operes el chafer sin agua; esto puede dañar el equipo y provocar riesgos.

### Cómo usar esta checklist en SeatPilot

1. **Versión impresa:** imprime el documento y compártelo con los clientes durante la fase de discovery (F0/F1).  
2. **Versión digital:** convierte cada fila en componentes de formulario dentro de `apps/web` o `apps/checkin` para capturar las respuestas y generar un VenuePack inicial.  
3. **Sincronización:** las respuestas pueden almacenarse en Supabase (`rsvp.event_requirements`) y servir como input de Seat Designer / Ops Live.  
4. **Localización:** utiliza este contenido como fuente para el bundle de traducción ES/EN cuando se habilite el modo de múltiples idiomas.

> Mantén este archivo sincronizado con `docs/handoff/SeatPilot_Master_v1.0.md` (sección Operación) y con el diccionario de términos SeatPilot para asegurar consistencia terminológica.
