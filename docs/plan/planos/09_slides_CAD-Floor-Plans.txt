Blocks
A block is a named group of objects that act as a single 2D or 3D objec t.  Use 
them to create repeated content (symbols, components) such as d oors,

Edit Blocks
By redefining a block, you 
automatically update all references to that block.
Tip: If the actual lineworkwithin 
your block is on the “0” layer, the lineworkwill display as whichever layer you place the block.  This allows for the same block to appear as different line weights.
